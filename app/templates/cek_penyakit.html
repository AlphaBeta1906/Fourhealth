How to Fill in Symptoms

  • 1. Open the symptom input page.
  • 2. On the form, you will see an input box labeled "Enter Symptoms".
  • 3. Click on the input box to open the list of disease symptoms.
  • 4. You can select one or more disease symptoms by:
    • Clicking on the symptom you want to select.
    • If you want to select more than one symptom, repeat the same process and the symptoms will be automatically added.
    • The selected symptoms will be displayed in the input box.
  • 5. If you want to change or remove the selected symptoms, you can reopen the input box and click or click again on the symptom you want to modify or remove.
  • 6. After filling in the form or any other page, click the "Check" button or to submit the data you have entered.
*Make sure to follow the instructions provided on the actual disease symptom input page.
